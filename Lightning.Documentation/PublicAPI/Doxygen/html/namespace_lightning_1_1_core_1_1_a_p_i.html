◆ RenderingBlendMode

enum Lightning.Core.API.RenderingBlendMode
strong

RenderingBlendMode

July 13, 2021

Defines the blending mode to use for rendering. Equivalent to SDL.SDL_BLENDMODE.

Enumerator
None 

No blending:

dstRGBA = srcRGBA

AlphaBlending 

Alpha blending:

dstRGB = (srcRGB * srcA) + (dstRGB * (1-srcA)) dstA = srcA + (dstA * (1-srcA))

dstRGBA = new Color4(dstRGB.R, dstRGB.G, dstRGB.B, dstA.A);

AdditiveBlending 

Additive blending:

dstRGB = (srcRGB * srcA) + dstRGB

ColourModulation 

Colour modulation:

dstRGB = srcRGB * dstRGB dstA = srcA;

dstRGBA = new Color4(dstRGB.R, dstRGB.G, dstRGB.B, dstA.A);

ColourMultiplication 

Colour multiplication:

dstRGB = (srcRGB * dstRGB) + (dstRGB * (1-srcA)) dstA = (srcA * dstA) + (dstA * (1 - srcA))

◆ ServiceImportance

enum Lightning.Core.API.ServiceImportance
strong

Non-DataModel

ServiceImportance enum (internal: only for SCM)

2021-04-03

Determines the importance of a service.

Enumerator
Low 

This service can crash without being immediately reinitialised. An example of this is the SettingsService, as it only requires to be called when settings are changed.

High 

This service requires the engine to quit, restart, or exit the current game. An example of this is the NetworkReplicator for network-based Lightning games; if it fails, the current game must be exited.
